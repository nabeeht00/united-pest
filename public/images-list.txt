HOSTING INSTRUCTIONS:
====================

1. Upload ALL files from the 'dist' folder to your hosting 'public_html' directory
2. Upload ALL image files from the 'public' folder to your hosting 'public_html' directory
3. Make sure image file names match exactly (including spaces and special characters)
4. Test each page to ensure images load correctly

IMAGE PATH FORMAT:
==================
All images use absolute paths starting with "/" 
Example: /Untitled design.png will load from https://yourdomain.com/Untitled design.png

IMPORTANT NOTES:
================
- Keep original file names exactly as shown
- Upload images to the root directory (public_html)
- Do not create subfolders for images
- Test the website after upload to verify all images appear

If any images don't appear after hosting:
1. Check file names match exactly
2. Verify images are in the root directory
3. Check file permissions (should be 644)
4. Clear browser cache and test again
